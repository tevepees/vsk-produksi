Deploy ke Cloudflare Pages

Web app ini terdiri dari 3 file saja. Deploy gratis ke Cloudflare Pages dalam ~10 menit. Setelah deploy, kamu dapat link seperti vsk-produksi.pages.dev yang bisa dibuka dari HP manapun.

1
Download file web app ini

Web app ini terdiri dari 3 file yang sudah terpadu:

• index.html — halaman utama
• manifest.json — agar bisa di-install di HP
• sw.js — agar bisa dipakai offline

Kamu bisa download semua file ini dari link yang diberikan Claude, atau buat folder baru dan copy-paste isinya.

2
Buat akun GitHub (gratis)

1. Buka github.com → klik Sign Up

2. Daftar dengan email → verifikasi email

3. Buat repository baru: klik "+" → New repository

4. Nama repo: vsk-produksi → pilih Public → klik Create

5. Upload 3 file (index.html, manifest.json, sw.js) via tombol Upload files

3
Hubungkan ke Cloudflare Pages

1. Buka pages.cloudflare.com → daftar gratis

2. Klik Create a project → pilih Connect to Git

3. Pilih repository vsk-produksi dari GitHub

4. Di bagian Build settings: biarkan semua kosong (tidak perlu build command)

5. Klik Save and Deploy → tunggu ~1 menit

6. Kamu dapat link seperti: vsk-produksi.pages.dev ✅

4
Install di HP operator (opsional tapi recommended)

Android (Chrome):

1. Buka link web app di Chrome
2. Tap menu ⋮ → "Add to Home screen"
3. Web app muncul seperti aplikasi biasa di home screen

iPhone (Safari):

1. Buka link di Safari
2. Tap ikon Share (□↑) → "Add to Home Screen"
3. Tap Add

5
Update jika ada perubahan

Jika ada perubahan di web app (misalnya tambah field atau ubah target):

1. Upload file baru ke GitHub (replace file lama)
2. Cloudflare Pages otomatis update dalam ~1 menit
3. Refresh di HP — selesai

Tentang Penyimpanan Data

📱 Versi ini (offline-first): Data tersimpan di HP/browser masing-masing operator. Data tidak otomatis terpusat.
